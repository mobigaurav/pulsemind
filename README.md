# PulseMind

PulseMind is a beautifully designed mindfulness and wellness app built for iOS and Apple Watch. It helps users track mood, manage stress, and build better mental health habits through:

- Daily Journaling with mood tagging
- Stress Trends using HealthKit (HR, HRV, Sleep)
- Guided Breathing Exercises with sounds
- Focus Timer for productivity
- Smart reminders and mood notifications
- Apple Watch support with mood syncing

## Technologies

- SwiftUI (MVVM architecture)
- HealthKit
- CoreData
- WatchConnectivity
- Charts
- Notifications
- Custom onboarding + splash screen
